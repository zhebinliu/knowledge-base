## 第一部分：一个神经元在做什么（10分钟）

### 别被"神经元"这个名字唬住

"神经元"听起来很高级，好像跟人脑有什么神秘联系。**忘掉这层联系**。今天讲的所谓"人工神经元"，就是一个数学函数。仅此而已。

它做的事用一句话能讲清楚：**接收几个输入，每个输入乘上一个权重，加在一起，再做一次非线性变换**。

我们看演示。

【看左边的"决定吃不吃火锅"演示】

我设计了一个非常具体的场景。你在决定今晚吃不吃火锅，三个因素影响你：
- 天气有多冷（x1）
- 朋友邀请的强烈程度（x2）  
- 钱包饱满程度（x3）

但这三个因素**对你的影响不一样**。你可能特别看重朋友邀请，那这个权重就大；你可能不太在乎天气，那这个权重就小。这些"在乎程度"就是**权重**（w1, w2, w3）。

最后还有一个东西叫**偏置**（b），它代表你"基础上有多想吃火锅"——有的人本来就爱吃，b就高；有的人本来就不爱吃，b就低（负数）。

### 神经元的计算公式

把这些拼起来就是神经元的全部计算：

```
z = w1·x1 + w2·x2 + w3·x3 + b
y = activation(z)
```

第一行就是加权求和。第二行的 activation 叫激活函数，负责做一个非线性变换——把任意大小的z压到0-1之间。

【看右边的可视化和滑块】

我把x1=7（天气挺冷）、x2=5（朋友一般邀请）、x3=6（钱包还行）。权重 w2=5（很在意朋友）、w1=3（在意天气）、w3=2（不太care钱包），偏置 b=-30（基础上不太想出门）。

算一下：3×7 + 5×5 + 2×6 + (-30) = 21 + 25 + 12 - 30 = **28**。

激活函数把28压到接近1的值（0.95）。**输出大于0.5，神经元被激活，决定吃火锅**。

【拖动滑块，比如把"朋友邀请"调到0】

看，朋友不邀请了，输出就掉到0.04，神经元不激活，不去吃了。

### 这就是全部

这个公式简单得像小学数学。但**深度学习的全部秘密**就藏在这里：

> **学习一个神经元 = 找到合适的权重和偏置**

- 你的"对各种因素的在乎程度"，本质上是从你过去的经验（训练数据）里学出来的。
- 一个识别猫的神经元，它的权重就是从几万张猫图里学出来的——"耳朵尖"这个特征的权重应该多大、"会喵喵叫"的权重应该多大。
- 一个判断邮件的神经元，权重对应"含有'中奖'这个词应该扣多少分"。

**所有AI的"知识"，都存储在权重里**。GPT-4的1.7万亿参数，就是1.7万亿个这样的权重。

### 为什么需要激活函数

这里多说一句关于激活函数。如果没有激活函数，整个网络就退化成一个线性函数——再多层也是线性。激活函数（Sigmoid、ReLU、GELU这些）让网络具有了**非线性表达能力**。

非线性是什么意思？就是它能画出弯弯曲曲的边界，而不只是直线。现实世界的概念边界几乎都是弯曲的，所以非线性是神经网络能work的关键。

## 第二部分：为什么要堆很多层（8分钟）

### 单个神经元的局限

一个神经元只能画一条直线。如果你的数据是猫和狗散布在空间里，能用一条直线分开，那一个神经元就够了。

但现实里几乎没有这种场景。猫和狗的特征会重叠、会缠绕，需要更复杂的边界。

怎么办？**把神经元堆起来**。

### 看分层网络演示

【点击"中等任务"标签】

这是一个5层的网络。最左边是输入层（接收数据），最右边是输出层（给出结果），中间是隐藏层。

注意每个神经元都连接着上一层的所有神经元。这种结构叫"全连接网络"或"MLP"。

**关键洞察来了**：每一层在干不同的事。

如果这是一个识别人脸的网络：
- **第1层**学边缘——竖线、横线、斜线
- **第2层**把边缘组合成纹理——眼角、嘴唇形状
- **第3层**把纹理组合成器官——眼睛、鼻子、嘴
- **第4层**把器官组合成脸的部分——眼睛+鼻子+嘴的相对位置
- **第5层**判断这是谁

**这就是"分层抽象"——浅层学具体，深层学抽象**。

【切到"复杂任务"标签】

GPT-4据说有100多层。可以想象它在做这种事：
- 浅层学字符和词的拼写规律
- 中层学短语、句法、语法
- 中深层学语义、上下文关系
- 深层学概念、推理、风格

这就是为什么大模型既能写诗又能写代码——**不同的层在做不同抽象层次的事，最终组合出复杂能力**。

### 一个改变历史的事实

讲到这儿，我要插一段历史。

2012年之前，做图像识别的标准做法是**手工设计特征**。一群博士花几年时间，研究怎么从图片里提取"边缘"、"纹理"、"颜色直方图"这些特征。这个过程叫**特征工程**，是当时CV领域最核心的工作。

2012年发生了什么？AlexNet在ImageNet比赛上拿了冠军。**它什么特征工程都没做**——直接把原始像素喂给一个深度神经网络，让网络自己学。结果错误率比第二名低了10个百分点。

这是什么概念？相当于一群博士干了10年的工作，被一个网络一夜之间淘汰了。

**AlexNet的胜利不是某个算法的胜利，是"让网络自己学特征"这个思路的胜利**。从那之后，深度学习接管了一切。

### 参数数量的爆炸

【看"PARAMETERS"那个数字】

注意这里有个细节。简单网络可能只有几十个参数。中等网络几百到几千。复杂网络可以到几百万、几十亿、上万亿。

每多一层、每加一些神经元，参数数量都呈倍数增长。这就是为什么大模型的训练那么贵——**参数越多，需要的算力和数据越多**。

我们后面第7讲会讲Scaling Law，正是因为发现"参数量×数据量×算力"和模型能力之间有数学关系，才有了今天的"大力出奇迹"路线。

## 第三部分：权重是怎么调出来的（10分钟）

讲到这儿，关键的问题来了：**这些权重一开始从哪来？**

答案有点反直觉：**随机初始化**。开始训练的时候，所有权重都是随机数。

那它怎么变成"正确"的权重？答案是**梯度下降**。

### 看梯度下降演示

【滚动到第三个演示，看那条曲线】

我们假设只有一个权重需要调（实际上GPT-4有1.7万亿个，但原理一样）。横轴是这个权重的取值，纵轴是模型在这个权重下的"损失"——预测得越离谱，损失越高。

**我们的目标**：找到让损失最小的那个权重。

【点击曲线左边某处】

我把起点设在这里。模型刚开始训练的时候，权重是随机的，损失很高。

【点击"开始下降"】

看那个小球。它在干什么？

**它感受脚下的坡度，往下坡方向走一步**。再感受坡度，再走一步。一直走到谷底。

这就是**梯度下降**的核心机制。

### 梯度是什么

"梯度"就是**坡度**——告诉你往哪个方向走、走多陡。

数学上，梯度是损失函数对权重的导数。但我们今天不讲数学，**你只要记住：梯度告诉你权重该往哪边调**。

权重的更新公式：
```
w = w - lr × gradient
```

其中 lr 是**学习率**，控制每一步走多远。

### 学习率的玄学

【拖动学习率滑块到很大，比如0.8，然后再点重置和下降】

看，学习率太大会怎样？小球**直接跳过谷底**，甚至越跳越远，永远到不了最低点。这叫"训练发散"。

【把学习率调到很小，比如0.05】

学习率太小呢？走得**特别慢**。要走几百步才能到谷底。在大模型场景下，每一步都要消耗算力，太慢就训练不起。

**学习率是炼丹师的核心玄学之一**。怎么调？经验+反复试。现实里有更复杂的优化器（Adam、AdamW）和学习率调度策略，自动调整学习率。但底层思想就是这个滑块。

### 局部最优陷阱

【把"损失函数形状"切到"双谷"】

现实里的损失曲面没那么友好。它可能有好几个谷。

【在右边的浅谷点击设起点，然后下降】

看，小球跑到右边那个浅谷就停下来了。但**真正的最低点在左边**——它陷在了"局部最优"里。

这是神经网络训练的另一个核心难题。解决方案有：
- **随机初始化多次**，看哪次能跳出陷阱
- **加噪声**（SGD的随机性其实是优势）
- **动量优化器**（带着惯性，能冲过浅谷）

【再切到"波浪"】

更复杂的曲面有几十个局部最优。GPT-4的1.7万亿维空间里，谁也不知道有多少陷阱。但实践证明，**深度网络往往能找到足够好的解，不一定是全局最优，但够用**。

### 反向传播：一句话讲清楚

我们前面只讨论了一个权重。但实际网络有几百万个权重，怎么同时调？

答案叫**反向传播算法**（Backpropagation）。

数学上很复杂，但思想可以一句话说清楚：

> **从损失出发，逆着网络一层一层倒推，每个权重该怎么调**。

打个比方：项目失败了，怎么找原因？从结果倒推——客户不满意是因为方案不对，方案不对是因为需求理解错了，需求理解错了是因为访谈没做好。**逐层归因**。

反向传播就是这个意思。损失大了，倒推每一层每个权重应该调多少。

### 把整个训练过程串起来

到这儿我们可以把神经网络训练的全流程串起来了：

1. **随机初始化**所有权重
2. 喂入一批训练数据
3. **正向传播**：数据从输入层走到输出层，得到预测
4. 计算预测和真实答案的差距，得到损失
5. **反向传播**：从损失出发，倒推每个权重的梯度
6. **梯度下降**：用梯度更新每个权重
7. 回到第2步，循环几百万次

**就这么简单**。这就是从1986年Hinton提出反向传播，到2012年AlexNet，到2017年Transformer，到2023年GPT-4，背后的同一套机制。

## 收尾（30秒）

今天这一讲，我们打开了神经网络的黑箱。回顾三个核心：

1. **一个神经元 = 加权求和 + 非线性变换**。所有AI的知识都存在权重里。
2. **分层堆叠 = 自动特征学习**。浅层学具体，深层学抽象，淘汰了手工特征工程。
3. **梯度下降 = 在损失曲面上找最低点**。这是训练所有神经网络的统一机制。

GPT-4和你电脑上的简单分类器，**底层机制完全一样**，只是规模不同。

下一讲我们讲深度学习革命的具体故事——从CNN到Transformer，理解为什么2017年那篇"Attention is All You Need"论文，开启了大模型时代。

## 备注：常见疑问

**Q: 神经网络真的像人脑吗？**
A: 在60年代提出时灵感来自人脑，但今天的神经网络和真实神经元差别巨大。最好把它理解成纯粹的数学结构，"神经"两个字就是个历史包袱。

**Q: 为什么一定要随机初始化？全部初始化为0不行吗？**
A: 不行。如果所有权重相同，反向传播时所有神经元会得到相同的梯度，永远学不出差异。随机初始化是为了打破对称性。

**Q: GPT这么大模型怎么训练？**
A: 几千张GPU并行训练几个月，分摊计算和存储。一次训练成本几千万到上亿美金。这个第7讲会展开。
