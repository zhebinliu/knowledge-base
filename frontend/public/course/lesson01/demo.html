AI到底是什么

从写规则到看例子
三十分钟，建立第一层直觉
我们用一个最简单的问题热身：怎么让计算机识别一只猫？ 这个问题看似幼稚，却恰好把AI发展史上的两条主要路线摊开在面前。 下面这个交互演示，你可以亲手试试两种方法的差别。
演示一：教计算机认猫
同一个任务，两种思路。切换标签亲手体验差别。
人类专家定义的规则
  • IF 有毛 AND 四条腿
  • AND 有尾巴 AND 有胡须
  • AND 尖耳朵 AND 体重<10kg
  • THEN 是猫

橙色=猫的特征空间，蓝色=非猫。模型从这些样本里学到一条分界线。

模型不背规则，它学的是一个连续的"猫味"评分函数。

规则系统的极限：老虎也满足"有毛+四条腿+尖耳朵"，会被判断成猫。 你可以一直加规则——"体重要<10kg"——但现实中的概念边界永远比你能写出的规则多一条。

学习系统的妙处：它从来不"理解"什么叫猫，它只是在拟合一条边界。 给它越多样本，这条边界画得越准。这就是机器学习时代数据=石油的本质。

演示二：AI的三大流派
点击卡片，看每条路线的核心信念和代表性工作。这三条线在过去七十年此消彼长。
PATH · 01
符号主义
Symbolism
智能 = 操作符号的能力。 只要把世界规则写清楚，机器就能像人一样思考。 1956到1980年代的主流。
代表：专家系统、逻辑推理、知识图谱
困境：知识难以穷举，复杂场景规则爆炸
PATH · 02
连接主义
Connectionism
智能 = 神经元之间的连接强度。 模仿大脑结构，让网络从数据中自己学。 2012年后绝对主流。
代表：神经网络、深度学习、Transformer
困境：黑箱、能耗高、需要海量数据
PATH · 03
行为主义
Behaviorism
智能 = 在环境中的反馈与适应。 不预设知识，只在试错中学策略。 强化学习的思想源头。
代表：AlphaGo、机器人控制、RLHF
困境：训练成本高，奖励函数难设计

今天的大语言模型，本质是连接主义的胜利，但融合了另外两派的精华： 用海量文本训练神经网络（连接主义）， 让它学会调用工具和符号推理（符号主义）， 再用人类反馈强化学习对齐（行为主义）。

理解了这一层，你就理解了为什么"AI不是魔法，而是函数拟合"—— 只是这个函数大到了7000亿参数。
