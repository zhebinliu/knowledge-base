## 一、项目背景

友发钢管是国内大型焊接钢管制造企业,年产能约 800 万吨,客户结构以经销商(占比 65%)、工程项目(占比 25%)、终端零售(占比 10%)三类为主,营销组织分布在华北 / 华东 / 华南 / 西南四大区。

当前业务存在以下痛点:

1. **经销商管理粗放**: 全国 1200 余家经销商,价格政策由各大区销售经理口头沟通,缺少线上审批流,经销商串货 / 跨区抢单时有发生
2. **工程项目跟进周期长**: 大型工程订单从立项到回款平均 14 个月,跨多个部门 (销售 / 技术 / 生产 / 财务) 协作信息散落在微信群、邮件、Excel,管理层无法实时掌握进度
3. **终端零售数据缺失**: 终端门店 80% 仍走线下点单 + 现金,门店级动销 / 库存 / 价格执行数据无法回传总部,导致总部促销策略难以精准下沉
4. **数据孤岛严重**: 现有 ERP (用友 NC) 只覆盖到出厂层,客户 / 订单 / 库存 / 财务数据未打通,经营分析依靠 IT 部门定期出报表,时效性差

## 二、项目目标

本项目为 CRM 一期,目标在 6 个月内完成以下能力建设:

1. 经销商全生命周期管理 (引入 / 准入 / 签约 / 价格政策 / 返利 / 退出),实现政策线上审批 + 串货预警
2. 工程项目机会管理 (线索 / 商机 / 项目 / 投标 / 合同 / 回款),实现 14 个月销售漏斗可视化
3. 终端门店动销采集 (轻量级移动端 / 微信小程序,覆盖 5000+ 终端门店,首期上线 1500 家)
4. 与 ERP 的客户主数据双向同步、订单数据单向同步 (CRM → ERP),报表层独立

## 三、范围与边界

### 包含范围

- 销售云核心模块: 客户 / 联系人 / 商机 / 合同 / 回款
- 渠道管理模块: 经销商档案 / 政策 / 价格管控 / 串货预警 / 返利计算
- 项目工程模块: 商机阶段管理 / 投标管理 / 项目组成员协作
- 移动端: iOS / Android / 微信小程序 (终端门店动销采集)
- 集成: 与用友 NC ERP 客户主数据 / 订单数据集成
- 报表与看板: 销售漏斗 / 经销商健康度 / 终端动销三大主题

### 不包含范围 (二期或后续)

- 服务云 (售后 / 工单)
- 营销云 (大众营销 / SCRM)
- BI 分析 (复杂多维分析)
- 与 OA (泛微) 的流程集成
